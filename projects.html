Tax Guide Query-Response Recommendion System

This project was a part of the EY Silver Badge AI Challenge. An interactive system for recommendations from TAX GUIDE was developed using BM25+ model. The data was pre-processed and bag of words approach was combined with the knowledge of n-grams and POS tags. A flask-python based website was deployed on the SAP cloud.

Lip Reading

This project aims at tracking the lip region based on colour and shape information. This project begins with the videos being shot focused basically to the lip area. The video thus obtained will be segmented to frames which are used for designing and implementing the experimental models. Further, the extracted images will be processed using MATLAB to correlate every lip image to its equivalent syllable. The final output of this system is the text display of the alphabet spoken by the speaker. The experimental results have been tested on different frames under constraints of illumination condition and head motion.

Home Automation System

This project is implemented using Raspberry Pi and the web application using Flask. The Raspberry Pi is connected to several LED’s, which can be switched on and off using buttons on the website. The temperature and humidity DHT11 sensor takes its readings and the values are constantly updated on the website. When the security system is enabled, an Ultrasonic sensor is used to detect any movement, with small distance readings triggering a buzzer.

Audio Video Studio

The project entitled “Audio Video Tapes” which keeps the record as a complete system for a music store. It maintains the database for the store, keeps the record of CD’s/DVD’s sale and purchase, demand, and provides information about the collection of the store. This is a comprehensive software suite consisting of integrated modules for various aspects of the music store. This software aims at reducing the workforce, management and widens the scope of the store. It increases the reliability and portability of database storage.
